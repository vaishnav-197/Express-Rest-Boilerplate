# Express-Rest-Boilerplate

---



#### Setup 
1. ###### fork the repository
2. ###### clone the repository
3. ###### move to folder
4. ###### `` npm install ``
5. ###### `` npm start ``
6. ###### `` npm start dev `` for developement server



### Packages Used 
1. Express
2. Morgan
3. Mongoose
4. Chalk
5. Cors
6. Body-Parser
7. Nodemon
